# TravelTrucks Frontend

Vite + React + Redux ile geliştirilmiş karavan kiralama frontend uygulaması.

## Özellikler
- Ana sayfa, katalog sayfası, karavan detay sayfası
- Backend ile entegrasyon (Axios)
- Filtreleme (backend-side query params)
- Favorilere ekleme (localStorage)
- Load More (pagination)
- Yorumlar ve rezervasyon formu (local demo)
- React Router, Redux Toolkit kullanımı

## Kurulum
1. `npm install`
2. `npm run dev`
3. Tarayıcıda `http://localhost:5173` açın.

## Deploy
Vercel veya Netlify kullanabilirsiniz. Build komutu: `npm run build`.
